# Steam Wishlist
This project allows to see games from users Steam Wishlist. By entering Steam User ID, it shows basic info about games in users wishlist, current discounts and prices in Russian Rubles and Kazakhstani Tenge.\
It consists of two apps:\
Backend app written on Python to gather wishlists information using Steam API.\
Frontend app written on React to allow user input of Steam User ID to gather wishlist and to show wishlist information.
